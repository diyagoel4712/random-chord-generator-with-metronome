## Random chord generator

This script generates a random chord every 4 beats and also plays a metronome. Ideal for practising a musical instrument!

You can tweak the chords in your arsenal, the tempo, or even the pitch of the metronome to make it more soothing.

Most required packages are basic, with the exception of simpleaudio.

Happy practicing :)

To run this script, call it from a command line with the following arguments:
- `-t` or `--tempo`: takes a positive integer value
- `--chords`: takes one or more string values from "major", "minor", "seventh" or "all"

You can modify the script to accept more arguments such as frequency (pitch) or sample rate, or you could change these values within the script (which is what I have done).
